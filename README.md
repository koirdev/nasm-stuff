# nasm-stuff
_Learning Assembly (NASM)_

# Compilation
---
- Windows (**win64**):

```nasm -f win64 main.asm -o main.o``` - (you can also use the "[**build.bat**](build.bat)" file)

# Linking

```ld main.o -o main.exe``` - (you can also use the "[**build.bat**](build.bat)" file if you are on Windows)
