# nasm-stuff
_Learning Assembler (NASM)_

# Compilation
---
- Windows (**win64**):

```nasm -f win64 main.asm -o main.o``` - (you can also use the "[**build.cmd**](build.cmd)" file)

# Linking

```ld main.o -o main.exe``` - (you can also use the "[**build.cmd**](build.cmd)" file if you are on Windows)
